# profile

> Nuxt.js project

## Build Setup

```bash
# install dependencies
# установите зависимости
$ npm install # Or yarn install

# serve with hot reload at localhost:3000
# запустите проект, приложение доступно по адресу http://localhost:3000
$ npm run dev

# build for production and launch server
# собрать приложение для продакшн-версии и запустить сервер в продакшн-режиме
$ npm run build
$ npm start

# generate static project
# собрать приложение и сгенерировать каждый маршрут в виде HTML файла
$ npm run generate
```

For detailed explanation on how things work, checkout the [Nuxt.js docs](https://github.com/nuxt/nuxt.js).
